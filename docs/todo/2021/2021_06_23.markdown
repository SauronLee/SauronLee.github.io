（１）授業について：
	データサイエンス宿題：論文（１）の考え：論文（１）のAHPの初期マトリックスは、学生のアンケート調査を使用しています。 アンケート調査では、学生は2つの特性の比較評価を行わず、一方的な評価のみを行った。 このように、学生の潜在意識は、2つの関心がない特性（例えば：家からの距離、休暇の長さなど）を比較しません。 したがって、AHPによって得られた重みは、学生の最も重要な特性の1つまたは2つにのみ反映されるため、AHPはその役割を果たしません。 さらに、企業の特性（特徴エンジニアリング）について、もうと特性（作業環境など）を追加する方が良いです（多次元に向け、でも手動ラベリングが難しい）。 または、人工的な特徴を完全に放棄し、埋め込みを使用して、それ自体で特徴を学習させます（Skip-Gram Model）。比較のために埋め込み＋レコメンダシステムの方法を使用することができます（例えば：成功したビジネスケースAirbnb、論文２）。（時間があれば自分が会社埋め込みを作りたい）
